# puppeteer-social-image-cli

<p align="center">Create dynamic social share images with <a href="https://github.com/chrisvxd/puppeteer-social-image">puppeteer-social-image</a> from the command line.</p>

```sh
yarn install
```

## Usage

Runs on node 10

```sh
yarn run psi
```
